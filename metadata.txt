changelog=

    Version 0.8:
    - ✨ Neue Funktion: "Versand" 
        - Im Projektodner wird ein "Versand"-Ordner erstellt
        - Ausgewähnlte Layer werden in einem GeoPackage gespeichert in diesem Ordner gespeichert
        - In dem Ordner wird eine Projektdatei erstellt, damit die Layer gleich aussehen wie im Projekt
        - Projektdatei und GeoPackage können zusammen versendet werden.
    - im Hintergrund werden Ressourcen anders verwendet (ohne Kompilieren)


    Version 0.7:
    - bugfix bei Erstellung eines neuen Projekt-GeoPackages 

    Version 0.6:
    - Beschriftung der Funktionen verbessert
    - bugfix: Layer, die nicht in einer Gruppe sind, werden nicht mehr umbenannt.
    
    Version 0.5:
    - Neue Kategorien für Layer: 
        - GeoPackage, aber nicht Projekt-GeoPackege
        - Kein GeoPackage (aber im Projektordner)

    Version 0.4.1:
    - Tool Tips der Symbole überarbeitet

    Version 0.4:
    - Symbole, die die Datenquelle der Layer anzeigen

    Version 0.3:
    - Beim Verschieben wird der Stil des Layers vom Ursprungslayer übernommen
    - Umbenennen kann rückgängig gemacht werden.

    Version 0.2:
    - erster Release
